# How to Add Images to Your TravelGo Website

## Step 1: Get Your Images

### Free Image Sources:
1. **Unsplash** (https://unsplash.com) - Search for:
   - "Indian train"
   - "Indian bus"
   - "Indian airline"
   - "Indian hotel"
   - "India travel"

2. **Pexels** (https://pexels.com)
3. **Pixabay** (https://pixabay.com)

### Recommended Searches:
- Hero background: "India travel", "Taj Mahal", "Indian tourism", "backpacking India"
- Bus: "Indian bus", "bus travel India", "Volvo bus India"
- Train: "Indian Railways", "train India", "Rajdhani Express"
- Flight: "Air India", "IndiGo plane", "Indian airport"
- Hotel: "Indian hotel", "hotel room India", "Taj hotel"

## Step 2: Download and Rename

Download 5 images and rename them exactly as:
1. `hero-bg.jpg` - Main background image
2. `bus-service.jpg` - Bus image
3. `train-service.jpg` - Train image
4. `flight-service.jpg` - Flight image
5. `hotel-service.jpg` - Hotel image

## Step 3: Save in the Correct Folder

Place all images in:
```
travelgo_project/static/images/
```

## Step 4: Restart Your App

If the app is running, restart it to see the new images.

## Note:
- Images will fall back to Unsplash URLs if local files are not found
- Recommended size: 800x600px for service images, 1920x1080px for hero
- Supported formats: .jpg, .jpeg, .png
- Keep file sizes under 500KB for faster loading
